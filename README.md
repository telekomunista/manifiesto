# manifiesto

Hackeando el manifiesto del partido comunista (Marx & Engels 1848) para modificarlo a las necesidades del colectivo telekomunista.

Empezamos con la traducción [de esta página](https://www.marxists.org/espanol/m-e/1840s/48-manif.htm), la cual fue digitalizada para el Marx-Engels Internet Archive por José F. Polanco en 1998. Retranscrito para el Marxists Internet Archive por Juan R. Fajardo en 1999.
